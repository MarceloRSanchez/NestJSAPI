## Description

Nest API project - Built on [Nest](https://github.com/nestjs/nest) framework.

## Installation

```bash
# Install CLI
$ npm i -g @nestjs/cli

# Install dependencies
$ npm install
```

## Running the app on Development mode

```bash
# For development and check use the watch mode
$ npm run start:dev

# For development but without hotreloading
$ npm run start
```

## Running the app on Production mode
```bash
# To use in production mode compile and init compiled project
$ npm run build
$ npm run start:prod
```
